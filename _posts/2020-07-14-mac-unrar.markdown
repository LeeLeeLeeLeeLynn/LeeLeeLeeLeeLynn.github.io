## UnRAR 

*UnRAR* is the name of two different command-line applications for extracting RAR file archives.

简单的命令行RAR解压工具——UnRAR。



## UnRAR下载 

使用homebrew下载

`brew install unrar`



## UnRAR使用 

cd 到 rar文件所在目录下，使用

`unrar x 文件名.rar` 解压。

解压后，文件在同一目录下。
