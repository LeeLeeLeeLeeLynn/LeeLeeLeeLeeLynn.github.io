## python SimpleHTTPServer :快速共享Linux系统的某个目录　

​		1）进入待分享的目录
　　2）执行命令python -m SimpleHTTPServer 端口号
　　　　注意：不填端口号则默认使用8000端口。
　　3）浏览器访问该主机的地址：[http://IP](http://ip/):端口号/

```bash
python -m SimpleHTTPServer 8080
```
